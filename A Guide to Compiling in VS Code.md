#  1. C++

## 1.0 Example of Code

```c++
# include <iostream>
using namespace std;

int main() {
    cout << "Hello World!";
    return 0;
}
```
## 1.1 Compiling

```bash
g++ main.cpp -o main.exe
```
## 1.2 Executing

```bash
./main.exe
```

```
Hello World!
```
